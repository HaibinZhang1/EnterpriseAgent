## 17.3 关键风险检查

| 风险 | 是否处理 | 设计措施 |
|---|---:|---|
| 客户端绕过权限 | 是 | 服务端每个主操作重新计算权限，客户端不作为权威 |
| 可见性泄露 | 是 | 服务端搜索和详情先按 VisibilityPolicy 过滤 |
| 本地文件写坏 | 是 | ExecutionPlan、备份、原子写、回滚 |
| LocalExecutor 伪代码不可编译 | 是 | 已修正 result 作用域，finally 使用 finalResult |
| MCP 密钥泄露 | 是 | SecureVariableStore 本地保存，日志/审计脱敏 |
| 包路径穿越 | 是 | SafeZipExtractor 拒绝 |
| tempUploadId 无生命周期 | 是 | 新增 temp_uploads 表和状态流转 |
| 下载统计口径不清 | 是 | download_tickets.purpose + metric_period_aggregates |
| AI 预审替代人工 | 是 | AI 只输出风险，不自动通过/拒绝 |
| 审计缺失 | 是 | 关键写操作审计同事务 |
| 重复审核 | 是 | revision 校验、条件更新和幂等键 |
| 下载凭证泄露 | 是 | 短期凭证、hash 保存、不记录明文 |
| 作者治理接口混入 admin | 是 | 新增 `/me/extensions/*`，与 `/admin/extensions/*` 分离 |
| 离线状态误用 | 是 | OfflinePolicy 限制新主操作 |
| 安全下架存量不可控 | 是 | 明确只处理托管项并说明离线限制 |
| Docker 部署半升级 | 是 | Flyway 失败服务端不启动 |
| 最后系统管理员被删除 | 是 | LastSystemAdminGuard 事务锁定检查 |
