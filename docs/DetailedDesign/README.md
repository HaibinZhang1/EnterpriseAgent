## 设计范围

- 客户端后端；
- 服务端；
- 模块划分、流程、接口、数据结构、依赖和实现思路；
- 异常处理、扩展点、配置管理和测试建议；
- 开源参考实现。

## 不包含

- 客户端前端 UI；
- Web 管理端前端 UI；
- 视觉和交互样式；
- Agent 执行内核。
