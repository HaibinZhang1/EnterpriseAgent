# 11. 管理端：概览

## 11.1 页面目标

概览页是 Web 管理端默认落点，用于管理员快速处理待办、发现异常和进入关键任务。

概览页只回答三个问题：

1. 当前有哪些待处理事项？
2. 当前有哪些风险或异常？
3. 我应该进入哪个管理任务？

## 11.2 数据范围

部门管理员看到：

- 本部门及下级部门待审核申请。
- 本部门及下级部门扩展状态。
- 本部门及下级部门用户状态。
- 本部门及下级部门相关审计摘要。
- 本部门及下级部门用户的客户端设备摘要。

系统管理员看到：

- 全局待审核申请。
- 全局扩展状态。
- 全局用户状态。
- 全局审计摘要。
- 客户端设备与更新状态。
- AI 系统预审状态。

## 11.3 页面结构

```text
概览
├─ 待办卡片
├─ 风险与异常
├─ 最近审核
├─ 最近审计
└─ 快捷入口
```

## 11.4 待办卡片

展示：

- 待审核申请数。
- 需修改申请数。
- 高风险申请数。
- AI 预审警告申请数。
- 异常扩展数。
- 冻结用户数。

系统管理员额外展示：

- 客户端更新待处理版本数。
- 最近更新失败事件数。
- 客户端版本过旧设备数。

## 11.5 风险与异常

展示最近需要关注的事项：

- 高风险发布申请。
- AI 预审失败或警告较多的申请。
- 审核超过建议处理时间的申请。
- 已下架但仍有大量本地使用的扩展。
- MCP 配置写入失败较多的扩展。
- Plugin 安装失败较多的扩展。
- 本地事件同步失败较多的扩展。
- 客户端版本过旧的设备。

## 11.6 最近审核

展示最近处理的审核记录：

- 扩展类型。
- 扩展名称。
- 提交人。
- 提交部门。
- 审核结果。
- 审核人。
- 时间。

点击后进入审核详情。

## 11.7 最近审计

展示最近关键审计事件：

- 部门与用户。
- 部门管理。
- 发布申请。
- 审核决定。
- 扩展上下架。
- 客户端设备。
- 客户端更新。

点击后进入审计详情。

## 11.8 快捷入口

部门管理员快捷入口：

- 审核待办。
- 扩展管理。
- 部门与用户。
- 审计日志。
- 客户端设备。

系统管理员快捷入口：

- 审核待办。
- 扩展管理。
- 用户与部门。
- 审计日志。
- 客户端设备。
- 客户端更新。
- 系统设置。

## 11.9 空状态

当没有待办时，显示：

- `当前没有待处理事项`。
- 快捷入口仍保留。

## 11.10 验收标准

- 管理端登录后默认进入概览。
- 部门管理员只能看到管理范围内数据。
- 系统管理员看到全局数据。
- 概览页不承载具体审核动作，只提供入口。
- 概览页可展示 AI 预审警告、MCP 接入失败、Plugin 安装失败和客户端设备异常入口。
