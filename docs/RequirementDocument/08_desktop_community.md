# 8. 桌面端：社区

## 8.1 页面目标

社区用于发现企业内部可用的 Skill、MCP Server 和 Plugin。

社区的默认落点是社区首页。社区首页展示搜索、榜单和类型入口；只有用户发起搜索或进入类型探索后，才展示结果卡片、分类和筛选。

## 8.2 社区入口规则

- 用户点击一级导航 `社区` 后进入社区首页。
- 若用户未登录，可先弹出登录框；登录成功后进入社区首页。
- 社区首页不展示结果卡片网格。
- 社区首页可以展示榜单条目，但榜单条目必须按类型分组。
- 已发布且未下架、未归档的扩展默认对登录用户展示卡片和详情。
- 勾选“仅授权范围内展示”的扩展只对授权范围内用户、作者和有管理权限的管理员展示。
- 勾选“仅授权范围内展示”的扩展不进入社区首页榜单统计。
- 授权范围控制安装、下载、复制配置、配置写入、新增启用和更新等主操作；停用、卸载和本地托管配置清理不因后续授权收缩而被禁止。

## 8.3 社区首页结构

```text
┌──────────────────────────────────────────────┐
│ 搜索区：搜索 Skill、MCP Server、Plugin          │
│ 按钮：搜索    + 发布                           │
├──────────────────────────────────────────────┤
│ Skill 区域：热榜 | Star 榜 | 下载量榜           │
│ MCP 区域：热榜 | Star 榜 | 使用量榜             │
│ Plugin 区域：热榜 | Star 榜 | 下载量/安装量榜    │
├──────────────────────────────────────────────┤
│ 类型入口：Skill 探索 / MCP 探索 / Plugin 探索    │
└──────────────────────────────────────────────┘
```

## 8.4 搜索区

### 文案

搜索框占位文案：

```text
搜索 Skill、MCP Server、Plugin、作者、部门或标签
```

### 行为

- 用户输入关键词并提交后，进入搜索结果页。
- 若关键词为空，点击搜索进入默认探索结果页。
- 搜索结果页默认展示所有类型分组结果。
- 用户可切换到 Skill、MCP Server、Plugin 单类型结果。
- 搜索提交时清空上一次结果页筛选条件。
- 搜索结果由服务端按可见选项过滤；前端不得通过隐藏字段自行决定是否展示。

## 8.5 发布按钮

社区首页右侧提供 `+ 发布` 按钮。

规则：

- 发布按钮不作为一级导航。
- 点击后打开发布向导。
- 未登录用户点击时先登录。
- 发布向导第一步必须选择扩展类型。
- 发布向导必须包含授权范围和可见选项。

## 8.6 榜单区

社区首页展示类型化榜单，不混排。

### 区域结构

社区首页必须包含三个榜单区域：

1. Skill 区域。
2. MCP Server 区域。
3. Plugin 区域。

每个区域横向排列三个榜单：

```text
热榜  |  Star 榜  |  下载量/使用量榜
```

### Skill 区域

- Skill 热榜。
- Skill Star 榜。
- Skill 下载量榜。

### MCP Server 区域

- MCP Server 热榜。
- MCP Server Star 榜。
- MCP Server 使用量榜。

说明：MCP Server 使用量固定使用“去重成功配置写入用户数”；连接检测次数、失败次数或复制配置次数只能作为管理端辅助指标，不得命名为使用量。

### Plugin 区域

- Plugin 热榜。
- Plugin Star 榜。
- Plugin 下载量/安装量榜。

说明：managed-package 展示去重成功安装用户数，config-plugin 展示去重成功配置写入用户数，manual-download 分开展示去重受控下载用户数和用户标记已安装数，并在榜单标题或 tooltip 中标明。

### 周期切换

- 热榜由公式计算，默认使用近 7 天活跃数据，不提供总榜、月榜、周榜切换。
- Star 榜支持总榜、月榜、周榜切换。
- 下载量/使用量榜支持总榜、月榜、周榜切换。
- 周榜按最近 7 天自然日窗口统计。
- 月榜按最近 30 天自然日窗口统计。
- 总榜按扩展发布以来累计有效数据统计。

### 榜单条目字段

每条榜单展示：

- 类型标识。
- 名称。
- 简短描述或分类。
- 作者。
- 部门。
- Star。
- 下载量、安装量或使用量。

### 点击行为

- 点击榜单条目打开对应类型详情。
- 点击“查看更多”进入对应类型结果页，并带入榜单类型和周期条件。

## 8.7 类型入口

社区首页提供三个入口：

1. Skill 探索。
2. MCP Server 探索。
3. Plugin 探索。

点击后进入对应类型结果页。

## 8.8 搜索结果页结构

搜索结果页包含：

- 返回社区首页入口。
- 当前搜索关键词。
- 类型切换。
- 类型化筛选器。
- 卡片网格。
- 排序。
- 空状态。

### 类型切换

提供：

- 全部。
- Skill。
- MCP Server。
- Plugin。

“全部”视图必须按类型分组：

```text
Skill 结果
MCP Server 结果
Plugin 结果
```

不得将三类结果直接混排。

## 8.9 Skill 结果页

### 筛选项

- 分类。
- 标签。
- 适用工具。
- 安装状态。
- 风险等级。

### 卡片字段

Skill 卡片展示：

- 图标或首字母。
- 名称。
- 描述。
- 分类。
- 最多 3 个标签。
- 作者。
- 部门。
- 版本。
- Star。
- 下载量。
- 本机安装状态。

### 卡片操作

- 点击卡片打开详情。
- Star / 取消 Star。

安装、更新、启用范围等动作放在详情中，不在卡片上堆叠。

## 8.10 MCP Server 结果页

- 适用工具。
- 权限等级。
- 风险等级。
- 本机接入状态。

### 卡片字段

MCP Server 卡片展示：

- 名称。
- 描述。
- 接入方式。
- transport。
- 适用工具。
- 作者。
- 部门。

### 卡片操作

- Star / 取消 Star。

安装到工具、复制配置和连接检测放在详情中，不在卡片上堆叠。

## 8.12 详情抽屉

所有类型详情使用统一抽屉模板，但内容按类型变化。

### 统一头部

展示：

- 类型。
- 名称。
- Extension ID。
- 当前版本。
- 作者。
- 部门。
- Star。
- 下载量、安装量或使用量。
- 状态。
- 授权范围。
- 可见选项。

### Skill 详情内容

- 概览。
- README。
- 兼容工具。
- 已启用位置。
- 版本历史。
- 审核与风险。

操作：

- 安装。
- 更新。
- 启用范围。
- 卸载。
- Star。

### MCP Server 详情内容

- 概览。
- 接入方式。
- transport。
- 配置示例。
- 变量说明。
- 权限声明。
- 数据访问说明。
- 连接检测说明。
- 版本历史。
- 审核与风险。

操作：

- 安装到工具。
- 连接检测。
- 编辑本地变量。
- 复制配置。
- 更新配置。
- 卸载配置。
- Star。

### Plugin 详情内容

- 概览。
- 目标工具。
- 安装模式。
- 兼容版本。
- 安装说明。
- 权限声明。
- 文件信息或配置清单。
- 版本历史。
- 审核与风险。

操作：

- 安装。
- 下载。
- 更新。
- 卸载。
- 查看安装说明。
- 标记已安装或已卸载，仅 manual-download 模式可用。
- Star。

### 未授权操作状态

当用户不在授权范围内但扩展仍可见时：

- 详情内容仍可阅读。
- Star 仍可使用。
- 安装、更新、下载、复制配置、配置写入、连接检测、自动写入等需要服务端授权的主操作置灰。
- 若本机已有托管内容，停用、卸载、移除托管配置和清理本地记录仍可使用。
- 置灰原因只保留一句话，不展开复杂说明。

### 不可见状态

当扩展勾选“仅授权范围内展示”且用户不在授权范围内时：

- 搜索结果不返回该扩展。
- 榜单不统计该扩展。
- 直接链接访问时显示简短提示：`该扩展不可见或不存在`。
- 不暴露扩展名称、作者、部门、描述和详情。

## 8.13 空状态

搜索无结果时：

- 显示当前搜索词。
- 显示当前类型。
- 提供“清空筛选”按钮。
- 提供“返回社区首页”按钮。

## 8.14 错误状态

错误包括：

- 企业服务不可用。
- 扩展不可见。
- 无权限安装。
- 无权限复制配置。
- 无权限下载受控文件。
- 无权限写入配置。
- 目标工具不支持该扩展。
- MCP 连接检测失败。
- Plugin 安装失败。
- 扩展已下架。
- 扩展已归档。
- 下载凭证获取失败。

错误提示必须给出下一步动作。

## 8.15 验收标准

- 点击社区后首先进入社区首页。
- 社区首页展示搜索、发布按钮、三类榜单区域和类型入口，不展示卡片网格。
- Skill、MCP Server、Plugin 各有热榜、Star 榜、下载量/使用量榜。
- Star 榜和下载量/使用量榜支持总榜、月榜、周榜切换。
- 提交搜索后才进入结果页。
- Skill、MCP Server、Plugin 在结果页和榜单中明确区分。
- “全部”结果按类型分组，不混排。
- 发布入口是按钮，不是一级页签。
- 勾选“仅授权范围内展示”的扩展对未授权用户不可见，且不进入榜单统计。
- MCP Server 详情可完成授权安装接入、配置写入、连接检测、更新和卸载。
- Plugin 详情可按安装模式完成受控安装、受控下载、更新、卸载或本地状态记录。
